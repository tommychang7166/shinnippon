## 📱 在 GoDaddy 上部署網站

### ✅ **步驟 1：準備文件**

1. **將 `shinnippon-website.html` 改名為 `index.html`**
   - 這樣 GoDaddy 才能正確識別為首頁

2. **確保文件夾中有：**
   - ✓ `index.html`（主網站）
   - ✓ `logo-white.png`（LOGO 圖片）

### ✅ **步驟 2：上傳到 GoDaddy（三種方法）**

#### **方法 A：使用 GoDaddy 檔案管理器（最簡單）**

1. 登入 GoDaddy 帳戶
2. 進入 **網站 → shinnippon.com.tw**
3. 點擊 **「檔案管理器」** 或 **「文件」**
4. 進入 **`public_html`** 或 **`www`** 資料夾
5. **上傳這三個文件**：
   - `index.html`
   - `logo-white.png`
   - `logo-black.png`（選擇性）
6. **完成！** 訪問 `shinnippon.com.tw` 即可看到你的網站

#### **方法 B：使用 FTP（更快速 - 適合大文件）**

1. 在 GoDaddy 取得 FTP 帳號密碼
2. 下載免費 FTP 工具：[FileZilla](https://filezilla-project.org/)
3. 在 FileZilla 中輸入：
   - **伺服器**：你的 FTP 位址（來自 GoDaddy）
   - **帳號**：你的 FTP 帳號
   - **密碼**：你的 FTP 密碼
   - **連接埠**：21
4. 連接後，進入 `public_html` 資料夾
5. 上傳文件
6. **完成！** 訪問 `shinnippon.com.tw`

#### **方法 C：使用 GoDaddy 網站編輯器（新手友善）**

1. 在 GoDaddy 的「發佈網站」頁面
2. 點擊 **「編輯網站」**
3. 在編輯器中上傳文件
4. **發佈網站**

## ⚙️ 自訂你的網站內容

網站中的這些部分可以輕鬆修改：

### **需要修改的內容**

在 `index.html` 中找到以下位置並修改：

#### 1️⃣ **聯絡資訊**
```html
<a href="mailto:info@shinnippon.com.tw" class="btn-contact">發送郵件</a>
<a href="tel:+886" class="btn-contact">撥打電話</a>
```
改為：
```html
<a href="mailto:你的郵箱" class="btn-contact">發送郵件</a>
<a href="tel:+886-你的電話號碼" class="btn-contact">撥打電話</a>
```

#### 2️⃣ **公司地址**
在聯絡區塊添加：
```html
<div class="contact-item">
    <h3>📍 公司地址</h3>
    <p>你的公司地址<br>台灣</p>
</div>
```

#### 3️⃣ **營業時間**

<div class="contact-item">

## 📞 遇到問題？

| 問題 | 解決方案 |
|-----|--------|
| LOGO 不顯示 | 確保 `logo-white.png` 在同一資料夾中 |
| 網站 404 錯誤 | 確保檔案改名為 `index.html` |
| 連結無效 | 更新郵箱和電話號碼到實際值 |
| 版面混亂 | 清除瀏覽器快取（Ctrl+F5） |
